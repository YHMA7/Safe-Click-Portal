## 📌 About the Project

**SafeClick Senior Portal** is a community service project developed as part of a university IT course. The goal is to bridge the digital divide for elderly citizens by providing a clean, accessible, and senior-friendly interface that consolidates essential online services in one place.

### 🎯 The Problem
Elderly citizens face significant barriers when using the internet:
- Complex, cluttered interfaces
- Small text and confusing navigation
- No clear path to essential services (health, government, bills)
- High vulnerability to online scams and fraud

### ✅ The Solution
A single-file HTML portal with:
- Large, readable text and high-contrast design
- Only the most essential services — no clutter
- Step-by-step instructions for each task
- Direct links to trusted Saudi government and service websites
- Built-in cybersecurity awareness tips

## ✨ Features

| Feature | Description |
|---|---|
| 🌙 **Dark / Light Mode** | Auto-detects system preference, with manual toggle |
| 🇸🇦 **Arabic / English** | Full RTL/LTR language switch |
| 🔤 **Adjustable Text Size** | A− / A+ controls for accessibility |
| 🔗 **Quick Access Links** | Direct links to WhatsApp, Absher, SEC, MOH, and more |
| 🚨 **Emergency Numbers** | All Saudi emergency numbers in one place |
| 🔍 **Search** | Filter services by keyword (Arabic & English) |
| 📱 **Responsive** | Works on desktop, tablet, and mobile |
| 🗂️ **Zero Dependencies** | Single `.html` file — no server, no frameworks needed |

## 🛠️ Services Covered

- 📹 **Video Call Family** — WhatsApp Web, Zoom, Telegram, Google Meet
- 🏥 **Book a Doctor** — Seha (صحتي), Mawid, MOH Portal
- 🏛️ **Government Services** — Absher, Najiz, Tawakkalna, Etimad, Muqeem
- 💳 **Pay a Bill** — SEC (كهرباء), NWC (مياه), STC, Mobily, Zain, STC Pay
- 🛡️ **Stay Safe Online** — NCA Cyber Awareness, CITC Consumer Protection
- 🚑 **Emergency & Help** — All Saudi emergency numbers + volunteer support

## 🚀 Getting Started

### Option 1 — Open Directly (Easiest)
```
1. Download index.html
2. Double-click the file
3. Opens in any browser — no installation needed
```

### Option 2 — Host on GitHub Pages (Recommended)
```
1. Fork this repository
2. Go to Settings → Pages
3. Set Branch: main → Save
4. Your site will be live at: https://YOUR-USERNAME.github.io/safeclick-portal/
```

### Option 3 — Host on Netlify (Drag & Drop)
```
1. Go to netlify.com and sign up (free)
2. Drag and drop index.html onto the deploy area
3. Your site is live instantly with a shareable link
```

## ♿ Accessibility

This portal was designed with senior accessibility in mind:
- **WCAG AA** compliant color contrast ratios
- **Keyboard navigation** supported on all interactive elements
- **Screen reader** friendly with `aria-label` and semantic HTML
- **Minimum 14px** base font size, adjustable up to 22px
- **RTL support** for Arabic language
